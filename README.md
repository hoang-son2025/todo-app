# Todo App

シンプルなブラウザ上の Todo アプリです。ローカルで index.html を開くだけで動作します。

## 概要
- タスクの追加、編集、選択、完了（Finish）、削除（Delete）が可能です。
- 複数タスクの一括操作はチェックボックスで選択して上部のボタンを使います。
- スマートフォン向けに全画面表示に対応しています。
- データはブラウザをリロードすると消えます（永続化は未実装）。

## 主な機能
- タスク追加：入力欄にテキストを入力して「Add Task」ボタンを押す。
- タスク編集：各タスクの「Edit」ボタンで編集モードに入る。編集後は Enter または Save ボタンで確定
- 選択：各タスク左のチェックボックスで選択可能。
- 一括完了（Finish）：チェックしたタスクを完了状態にする（完了後もチェックは維持されます）。
- 一括削除（Delete）：チェックしたタスクを一括で削除。
- ページネーション：タスクが多い場合はページ分割して表示します（1ページあたりの件数はコード内で設定）。

## 使い方
1. リポジトリをローカルにクローンまたはダウンロードします。  
   例: c:\Users\xxx\Desktop\Github\todo-app\ に配置済みであればそのままで OK。
2. ファイルをブラウザで開きます：  
   - ファイルエクスプローラーで index.html をダブルクリック、またはブラウザで `file:///c:/Users/xxx/Desktop/Github/todo-app/index.html` を開く。
3. タスクを追加する：
   - 入力欄にタスク名を入力し「Add Task」を押す。
   - タスクが1件以上あると上部に「Finish」「Delete」ボタンが表示されます。
4. 複数タスクを操作する：
   - 各タスク左のチェックボックスにチェックを入れる。
   - チェックした状態で上部の「Finish」または「Delete」を押す。
5. タスクを編集する：
   - 編集したいタスクの「Edit」を押し、内容を変更して Enter（または Save）で確定。

## 注意事項
- 現在はブラウザのメモリ上にのみデータが保持されます。ページリロードでデータは消えます。
- フルスクリーン表示やホーム画面追加の挙動はブラウザ／端末によって異なります。
